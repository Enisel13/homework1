Добре е преди да пушваш, да прегледаш статуса на текущите промени, за да провериш дали случайно не си променила нещо,
което не си искала да променяш

1. git status - с тази команда мога да видя какъв е статус на промените 

2. git add .  - Добавяш текущите промени като пакет за да могат да бъдат къмитнати. "." знакът точка означава да вземе
всичките промени, намиращи се в текущата директория(папка)

3. git commit -m "some descriptive message on your choice"

4. git push - пушваш къмита, който се е образувал от горната команда, а именно git commit -m "something", към github
